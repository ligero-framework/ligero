# Ligero Documentation

Documentation site for the [Ligero web framework](https://github.com/ligero-framework/ligero),
built with Docusaurus.

```bash
npm install
npm start        # dev server at http://localhost:3000
npm run build    # static site in build/
```

Deploys to GitHub Pages automatically on push to `main` (see `.github/workflows/deploy.yml`).
Content lives under `docs/`; the Spanish translation will be added via Docusaurus i18n.
